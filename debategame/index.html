Firebase 실시간 서버 연동

친구들과 함께 온라인 플레이를 즐기려면 본인의 Firebase 프로젝트 연결이 필요합니다.

💡 간단한 연동 방법 (무료)

  1. Firebase 콘솔에 접속하여 새 프로젝트를 생성합니다.
  2. 웹 앱(</>)을 추가한 후 생성되는 firebaseConfig 오브젝트를 복사합니다.
  3. 아래 입력란에 복사한 객체(또는 JSON 포맷)를 붙여넣고 연결하기를 누르세요!
🛠️ 고급 환경 변수 연동 지원: 로컬 폴더에 env.js 파일을 만들고 아래처럼 설정해 두면 이 창을 거치지 않고 자동 로그인됩니다! 
window.env = { apiKey: "...", projectId: "..." };
※ 입력하신 설정값은 서버로 전송되지 않고 본인의 브라우저 로컬 저장소(localStorage)에만 안전하게 기기 내부 보관됩니다.
